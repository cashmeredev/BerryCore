# Python 3.11.10 for BerryCore

Python 3.11 interpreter with full standard library and pip support for QNX/BlackBerry 10.

## Installation

```bash
qpkg install python3
```

## Usage

```bash
# Run Python
python3

# Check version
python3 --version

# Install packages with pip
python3 -m pip install requests
```

## Included Tools

- `python3` / `python3.11` - Python interpreter
- `pip` - Package installer (use: python3 -m pip)
- `idle3` - Python IDE
- `pydoc3` - Documentation tool
- `2to3` - Python 2 to 3 code converter
- `python3-config` - Python configuration tool
